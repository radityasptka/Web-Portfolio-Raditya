Web Pemesanan Hotel

Website ini dirancang untuk memudahkan pengguna dalam mencari dan memesan hotel dengan cepat dan efisien. Halaman utamanya menampilkan daftar hotel lengkap dengan gambar, nama, lokasi, dan harga per malam, serta dilengkapi fitur pencarian untuk menemukan hotel yang sesuai kebutuhan.

Web Top Up Games

Web ini dibuat untuk membantu pengguna melakukan top-up game dengan mudah. Halaman utamanya menampilkan daftar game populer dengan gambar dan tombol top-up, sehingga pengguna dapat memilih game dan melakukan transaksi dengan cepat.

Web Cafe

Website ini berfungsi sebagai platform promosi untuk kafe, menampilkan menu makanan dan minuman populer dengan desain yang menarik dan ramah pengguna. Halaman utamanya memuat daftar menu lengkap dengan gambar, nama, dan harga untuk menarik pelanggan.
